# Rsbuild Project

## Setup

Prepare the project:
```bash
pnpm add vuex@next vue-router@4 vue-i18n@10 -D
pnpm add @nutui/nutui @nutui/icons-vue -D
pnpm add axios -D
```

Install the dependencies:

```bash
pnpm install
```

## Get Started

Start the dev server:

```bash
pnpm dev
```

Build the app for production:

```bash
pnpm build
```

Preview the production build locally:

```bash
pnpm preview
```
